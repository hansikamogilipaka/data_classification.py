# Data Classification Using AI

## 📌 Overview
This project demonstrates supervised machine learning classification using the Iris dataset.  
It includes Logistic Regression, SVM, KNN, and Decision Tree models, with accuracy comparison and visualizations.

## ⚙️ Features
- Data preprocessing with StandardScaler
- Multiple classification models
- Confusion matrix heatmap
- Cross-validation
- Model saving/loading with Joblib
- Feature importance visualization
